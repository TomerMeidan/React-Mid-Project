# React Mid Project - User Management

## Overview

This project will simulate a connection to a database through REST API calls to https://jsonplaceholder.typicode.com. The frontend will be built using React and will allow for CRUD (create, read, update, delete) actions on the data.

## Setup

- Initialize a React app
- Install Axios for making API requests

## Features 

- Fetch list of resources from API
- Display resources in a list/table 
- Add new resources 
- Edit existing resources
- Delete resources

## API Integration

- Make a GET request to `https://jsonplaceholder.typicode.com/todos` to retrieve a list of todos
- Make a POST request to `https://jsonplaceholder.typicode.com/todos` to create a new todo
- Make a PUT request to `https://jsonplaceholder.typicode.com/todos/:id` to update a todo 
- Make a DELETE request to `https://jsonplaceholder.typicode.com/todos/:id` to delete a todo
